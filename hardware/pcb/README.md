# PCB del robot

Esta carpeta contiene la documentación disponible de la placa electrónica del robot.

## Contenido
- `source/`: archivo fuente del esquemático
- `images/esquematico.png`: captura del esquema eléctrico
- `images/pcb_fabricada.png`: fotografía de la PCB fabricada
- `images/pcb_montada.png`: fotografía de la PCB con componentes montados

## Estado
Actualmente se dispone del archivo fuente del esquemático y evidencia fotográfica de la placa fabricada y ensamblada.

## Componentes principales
- ESP32 DevKitC
- TB6612FNG
- LM1584EN

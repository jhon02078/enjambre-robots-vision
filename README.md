\# Enjambre de robots con visión por computadora



Sistema distribuido para control y monitoreo de múltiples robots móviles usando:



\- PC como servidor de visión y control

\- Raspberry Pi como nodo de cámara/stream

\- ESP32 en cada robot para actuación

\- Marcadores ArUco para localización en el workspace



\## Estructura del proyecto



\- `pc/vision\_server/`: aplicación principal de visión y control

\- `raspberry/camera\_stream/`: servidor MJPEG de la cámara

\- `esp32/`: firmware de cada robot

\- `docs/`: diagramas, fotos y documentación
